## 一、产品概述

### 1.1 什么是逆流选股

**逆流选股**是一套基于"资金行为+筹码分布+赔率"的中线趋势预判系统。

通过识别主力吸筹行为，在行情启动前入场，在共识形成前离场，获取超额收益。

**与短线择时的根本区别**：

| | 逆流选股 | 短线择时 |
|--|---------|---------|
| **持股周期** | 中线（数周~数月） | 短线（1~5天） |
| **信号来源** | 资金流+筹码集中度 | 技术指标+情绪 |
| **抗干扰能力** | 强（忽略短期波动） | 弱（突发事件易失效） |
| **年操作次数** | 10~20只 | 50~100只 |

### 1.2 核心优势

- **提前量**：在趋势形成前介入，不追涨
- **高赔率**：每笔操作要求上涨空间/止损幅度 > 3:1
- **自动化**：每周五18:00自动运行，人工<30分钟/周
- **可迭代**：三步逻辑层层过滤，信号质量可量化评估

### 1.3 版本历史

| 版本 | 日期 | 核心变化 |
|------|------|---------|
| v1.0 | 2026-05 | 五层漏斗筛选 + AI公告分析 |
| **v2.0** | 2026-05-13 | 三步逻辑重构 + 四维评分体系 |

## 二、设计思想

### 2.1 什么是「逆流」

**逆流的本质是：在趋势尚未形成时入场，在共识形成前离场。**

> 想象一条河。当水还在平静时（趋势酝酿期），你逆流而上，占据有利位置。等大多数人看到浪潮时（趋势形成），你已经身在浪尖。

### 2.2 逆流的四个原则

**1. 主动入市，不跟趋势**
大多数投资者看到机会时，机会已经被定价。逆流要求在别人还没看到时就已经布局。

**2. 高确定性再出手**
逆流不是每天都找机会。每年真正值得出手的股票可能只有 10~20 只，但每一只都要有清晰的逻辑。**宁可错过，不可错入。**

**3. 忽略短期，拥抱中线**
一旦逆流信号确认买入，就忽略短期波动。止损线放宽（10~15%），让趋势自然走出来。

**4. 赔率优先，趋势为王**
每一笔逆流操作，都要回答两个问题：
- **赔率够不够？** → 潜在涨幅/止损幅度 > 3:1
- **趋势成不成？** → 有没有催化剂驱动趋势启动

两个都满足，才操作。

### 2.3 行情启动阶段的定义

**逆流将"行情启动阶段"定义为：吸筹的中后期。**

这个阶段的特点：
- 主力已经完成大部分吸筹工作
- 均线收敛（多空收敛）
- 价格运行在箱体内（有主力干预证据）
- 距离正式启动还有一段距离，有介入机会

### 2.4 逆流的核心逻辑链

```
利空不跌 → 主力吸筹 → 筹码集中 → 技术蓄势 → 高赔率入局
```

## 三、三步选股逻辑

### 3.0 概览

```
第一步：现象发现（量化筛选）
  └ 主力连续流出 + 均线收敛 + 箱体稳定 + 背离度

第二步：成因归类（AI分析）
  └ 分析公告/研报 → 归入保留类型/排除

第三步：行情预测（AI四维评分）
  └ 催化剂 + 空间 + 持续性 + 筹码结构 → 确定性排序
```

---

### 第一步：现象发现

**目标**：找到具有连续资金流出但股价保持稳定的股票

#### 核心指标

| 指标 | 说明 | 阈值 |
|------|------|------|
| **主力净流出** | 近20日累计主力资金净流出 | 按市值分档 |
| **均线收敛度** | MA5与MA20的差距 | < 5% |
| **箱体宽度** | 近20日高低点差距/当前价 | < 15% |
| **背离度** | 实际跌幅 vs 预期跌幅 | > 50% |
| **边界触碰** | 价格触碰箱体边界次数 | ≥ 3次 |

#### 市值分档阈值

| 股票规模 | 流通市值 | 净流出阈值 |
|---------|---------|-----------|
| 小盘股 | < 50亿 | > 流通市值的 3% |
| 中盘股 | 50~200亿 | > 流通市值的 2% |
| 大盘股 | 200~1000亿 | > 流通市值的 1% |
| 超大盘 | > 1000亿 | > 流通市值的 0.5% |

#### 筛选条件（必要）

```
1. 主力资金近20日累计净流出 > 流通市值的对应比例
2. 均线收敛（MA5与MA20差距 < 5%）
3. 箱体宽度 < 15%
4. 价格在 MA20 上方运行
5. 背离度 > 50%
6. 箱体边界触碰 ≥ 3次
```

#### 指标计算公式

```
均线收敛度 = |(MA5 - MA20) / MA20| + |(MA10 - MA20) / MA20| × 100%

箱体宽度 = (近20日高点 - 近20日低点) / 当前价 × 100%

背离度 = (实际跌幅 - 预期跌幅) / 预期跌幅 × 100%
预期跌幅 = 主力净流出 / 日均成交额 × 观察日数 × 市场基准波幅(2%)
```

### 第二步：成因归类

**目标**：分析现象背后的原因，将其归类

#### 保留类型（可进入第三步）

| 类型 | 特征 | 中长线逻辑 |
|------|------|-----------|
| **主力吸筹** | 主力在低位悄悄收集筹码 | 建仓完成后有拉升需求 |
| **换庄** | 老庄派发，新庄接盘 | 新庄成本高，有动力做行情 |
| **业绩拐点** | 营收/净利润增速出现拐点 | 基本面驱动，中长线确定性强 |
| **政策驱动** | 行业/公司迎来政策利好 | 政策红利释放，趋势可持续 |
| **事件催化** | 并购重组、股权变更等 | 事件驱动型机会，爆发力强 |
| **估值修复** | PE/PB处于历史低位 | 价值回归是大概率事件 |

#### 排除类型（直接剔除）

| 类型 | 排除原因 |
|------|---------|
| **短线题材炒作** | 无基本面支撑，纯情绪推动，不适合中长线 |
| **长线资金调仓** | ETF/指数基金被动卖出，行情无持续性 |
| **主力对倒** | 自己左手倒右手，非真实吸筹，无拉升动力 |
| **解禁压力未消化** | 卖压尚未出清，入场即被套 |
| **基本面恶化** | 不应逆流抄底，逻辑不成立 |
| **纯技术性反弹** | 无逻辑支撑，反弹后继续寻底 |
| **主力高控盘** | 高度控盘后可能快速派发，风险大于机会 |

#### AI分析维度

```
1. 减持进度：是否有大股东/特定股东减持？是否接近尾声？
2. 业绩与基本面：业绩趋势如何？是否有超预期可能？
3. 行业政策：公司所在行业是否受益于近期政策？
4. 资金用途：若有大股东减持，减持资金用途是什么？
5. 高管动态：近期高管是否有增持/减持动作？
6. 限售解禁：近期是否有大额限售解禁？
7. 技术面：当前技术形态是否支持拉升？
```

### 第三步：行情预测

**目标**：在中长线上找成长性最确定的股票

#### 四维评分框架

```
成长确定性 = f(催化剂强度, 上涨空间, 趋势持续性, 筹码结构)
```

每维 0~25 分，综合 0~100 分。

#### 1. 催化剂强度（0~25分）

| 信号 | 得分 |
|------|------|
| 供给侧利空出尽（减持完成、解禁落地） | 5分 |
| 业绩拐点出现（营收/净利润增速拐点） | 5分 |
| 政策驱动（行业/公司迎来政策利好） | 5分 |
| 事件催化（并购重组、股权变更等） | 5分 |
| 无明显催化剂 | 0分 |

#### 2. 上涨空间（0~25分）

| 信号 | 得分 |
|------|------|
| 当前价距前高 > 20%，空间未被预支 | 10分 |
| PE/PB处于历史低位（<30%分位） | 8分 |
| 所属板块整体估值中枢上移 | 7分 |
| 空间一般（10~20%） | 5分 |
| 空间不足（<10%） | 0分 |

#### 3. 趋势持续性（0~25分）

| 信号 | 得分 |
|------|------|
| 均线多头排列（MA5 > MA10 > MA20） | 8分 |
| 成交量配合（上涨放量、回调缩量） | 7分 |
| 主力净流出放缓/转净流入 | 10分 |
| 部分满足 | 各给一半分 |
| 不满足 | 0分 |

#### 4. 筹码结构（0~25分）

| 信号 | 得分 |
|------|------|
| 股东户数持续下降 > 15% | 10分 |
| 股东户数下降 10~15% | 7分 |
| 股东户数下降 5~10% | 4分 |
| 股东户数持平或增加 | 0分 |

#### 确定性等级

| 综合得分 | 确定性等级 | 入池决策 |
|---------|-----------|---------|
| > 75分 | 高 | ✅ 入池 |
| 50~75分 | 中 | ✅ 入池 |
| < 50分 | 低 | ❌ 不入池 |

## 四、运行机制

### 4.1 周频运行

```
运行时间：每周五 18:00（收盘后）
运行频率：每周一次（不每日运行）
```

### 4.2 完整运行流程

```
18:00 第一步自动运行 → 妙想MCP筛选 → 候选列表（20~50只）
18:10 第二步AI归类 → 归类类型（6保留/7排除）
18:20 第三步AI评分 → 四维综合得分 → 排序
18:25 入池结果推送 → 飞书通知
```

### 4.3 各步骤自动化程度

| 步骤 | 自动化程度 | 人工介入 |
|------|-----------|---------|
| 第一步：现象发现 | 100%（系统自动） | ❌ 无 |
| 第二步：成因归类 | ~90%（AI分析） | ✅ 仅异常情况 |
| 第三步：行情预测 | 100%（AI评分） | ❌ 无 |

**总人工时间：< 30分钟/周**
