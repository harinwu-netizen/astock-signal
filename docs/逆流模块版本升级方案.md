## 三、详细改动说明

### 3.1 第一步：mx_nixflow_selector.py（P0）

#### 改动内容

**新增指标**：
1. **均线收敛度** = (MA5-MA20)/MA20绝对值 + (MA10-MA20)/MA20绝对值
2. **箱体宽度** = (近20日高点-近20日低点)/当前价 × 100%
3. **背离度** = (实际跌幅-预期跌幅)/预期跌幅 × 100%

**修改阈值**：
- 原：`amplitude_max=20.0`（振幅上限）
- 改：箱体宽度 < 15%（更精准）
- 原：`bottom_rise_min=0.0`
- 改：保留，作为加分项

**新增筛选条件**：
```
必要条件：
  ├ 主力净流出 > 市值对应比例
  ├ 均线收敛（MA5与MA20差距<5%）
  ├ 箱体宽度 < 15%
  ├ 价格在MA20上方
  └ 背离度 > 50%

加分项（辅助判断）：
  ├ 箱体边界触碰≥3次
  ├ 量能萎缩
  └ 均线粘合（多条均线缠绕）
```

#### 输出变化

原有 `NixFlowCandidate` 新增字段：
- `ma_convergence`: 均线收敛度
- `box_width`: 箱体宽度（%）
- `divergence_ratio`: 背离度（%）
- `box_touches`: 边界触碰次数

### 3.2 第二步：ai_analyzer.py（P0）

#### 改动内容

**Prompt 重写**：
- 原：7维度开放式分析 → 概率+入池建议
- 改：**结构化归类** → 类型判定 + 排除原因 + 逻辑支撑

**归类输出**：
```python
@dataclass
class NixFlowClassResult:
    code: str
    name: str
    scan_date: str

    # 归类结果
    类型: str                    # 主力吸筹/换庄/业绩拐点/政策驱动/事件催化/估值修复
    逻辑支撑: List[str]          # 3~5条核心证据
    中长线确定性: str            # 高/中/低

    # 排除判定
    是否排除: bool
    排除原因: str               # 若为排除类型

    # 下一步
    建议进入第三步: bool
```

**Prompt 设计要点**：
1. 先判断是否属于7种排除类型之一 → 若是，直接输出排除原因，skip
2. 再判断属于6种保留类型中的哪一种
3. 综合给出确定性评分（高/中/低）
4. 保留异常标注机制（AI不确定时标记"需人工确认"）

#### 人工介入方式

```
AI 归类结果 → 人工快速浏览确认
  ├ AI 标注"需确认" → 人工重点审核
  ├ 排除类型 → 人工确认后出池
  └ 保留类型 + 确定性"高/中" → 自动进入第三步
```

### 3.3 第三步：pool_manager.py（P0）

#### 改动内容

**评分逻辑重写**（从 AI定性 → 四维量化）：

```python
def calc_certainty_score(
    candidate,        # NixFlowCandidate（含第一步指标）
    class_result,     # NixFlowClassResult（第二步归类结果）
) -> NixFlowScore:
    """
    四维评分：
    1. 催化剂强度（0~25分）
    2. 上涨空间（0~25分）
    3. 趋势持续性（0~25分）
    4. 筹码结构（0~25分）

    综合得分：0~100
    输出：确定性等级（高/中/低）+ 入池排名
    """
```

**四维详细说明**：

| 维度 | 指标 | 满分 |
|------|------|------|
| 催化剂强度 | 利空出尽/业绩拐点/政策/事件 | 25分 |
| 上涨空间 | 前期高点距离/估值分位/板块对比 | 25分 |
| 趋势持续性 | 均线多头/成交量配合/主力净流入 | 25分 |
| 筹码结构 | 股东户数变化/主力控盘程度 | 25分 |

**输出格式**：

```python
@dataclass
class NixFlowFinalResult:
    code: str
    name: str

    # 综合结果
    综合得分: float              # 0~100
    确定性等级: str              # 高(>75)/中(50~75)/低(<50)

    # 四维分项
    催化剂得分: float
    空间得分: float
    持续性得分: float
    筹码得分: float

    # 归类
    股票类型: str               # 来自第二步

    # 核心结论
    核心结论: str               # 一句话总结

    # 入池决策
    入池建议: bool
    入池排名: int               # 按得分排序
```

**入池决策**：
- 每周取四维综合得分 TOP 3~5 入池
- 确定性等级为"低" → 不入池
- 已在池中的股票：得分持续下降或变为"低" → 出池
