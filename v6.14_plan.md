## 一、核心改动 · 资金流从"前置否决"改为"触发后验证"

### 现状（v6.13）
```
每轮扫描（15 分钟 × 14 次/天 × 7 只 = ~98 次/天）
  └─ 每只股票都调 get_money_flow()
     └─ 妙想数据池配额 ~60 次/日
     └─ 13:16 触限 → screener 兜底 4h47m → push2delay 兜底 1m
```

### 改造后（v6.14）
```
每轮扫描（98 次）
  └─ 只算技术指标，不调资金流

仅当决策 = BUY 时（预计 ~5 次/天）
  └─ 调 get_money_flow() 验证
     └─ 资金面正 → 执行 BUY
     └─ 资金面负 → 标记"技术BUY资金否决"，记入 evolution 负样本

STOP_LOSS / TAKE_PROFIT：直接执行，不验证资金流
```

### 改动 2: `indicators/signal_unified.py` · 新增 BUY 验证函数

**位置**: 新增函数（在 `analyze_unified` 之后）

```python
def verify_buy_with_money_flow(code: str, name: str) -> tuple[bool, str, Optional[MoneyFlowData]]:
    """
    BUY 信号的资金流验证（v6.14 新增）

    Returns:
        (passed, reason, mf_data)
        - passed=True: 资金面验证通过
        - passed=False: 资金面背离，记录为"假信号"
    """
    try:
        mf = get_money_flow(code, name)
        if mf is None:
            # v6.13 行为：保留 BUY，附"资金流缺失"warning（保守降仓）
            return True, "⚠️资金流缺失(降仓50%)", None

        if mf.main_net > 0 or mf.big_net > 0:
            return True, f"资金面验证通过({mf.signal})", mf
        else:
            return False, f"资金面否决: 主力净流出{mf.main_net:+.0f}万", mf
    except Exception as e:
        logger.warning(f"[MoneyFlow] 验证异常: {e}")
        return True, "⚠️资金流异常(降仓50%)", None
```

### 改动 3: `main.py` · `_handle_buy_signals` 流程改造

**位置**: `main.py:284` 附近的 `_handle_buy_signals`

**当前**:
```python
for code, signal in buy_signals.items():
    # 直接进风控检查 + 下单
    execute_buy_with_checks(code, signal)
```

**改为**:
```python
for code, signal in buy_signals.items():
    # v6.14: 资金流按需验证
    passed, mf_reason, mf_data = verify_buy_with_money_flow(code, signal.name)

    if not passed:
        # 技术 BUY + 资金否决 → 记入 evolution 负样本
        log_to_evolution({
            'code': code,
            'name': signal.name,
            'decision': 'BUY',
            'rejected_by': 'money_flow',
            'reason': mf_reason,
            'mf_data': mf_data.__dict__ if mf_data else None,
        })
        logger.info(f"[{signal.name}] BUY 被资金流否决: {mf_reason}")
        continue

    # 资金面 OK，执行买入
    signal.notes = mf_reason  # 附上验证结果
    execute_buy_with_checks(code, signal)
```
